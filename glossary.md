# Glossary

This glossary defines common terms used in transaction lifecycle descriptions.

## Transaction
A signed instruction sent to the blockchain to perform an action.

## Wallet
Software that allows users to create, sign, and send transactions.

## Node
A participant in the blockchain network that receives and shares transactions and blocks.

## Block
A collection of transactions that is added to the blockchain.

## Confirmation
The point at which a transaction is included in the blockchain and recognized by the network.
